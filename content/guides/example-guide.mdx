# Example Guide

This is an example guide file created in the **Guides** collection.

## URL Structure

This file will be accessible at: `/guides/example-guide-how-to-use-collections`

## What Are Guides For?

Guides are perfect for:
- How-to articles
- Reference documentation
- Setup instructions
- Process walkthroughs

## Step 1: Create Your Content

Navigate to the Guides collection in TinaCMS and click "Create New."

## Step 2: Fill in the Fields

Complete all required fields including title, excerpt, date, and author.

## Step 3: Write Your Content

Use MDX to write your guide content with formatting, code, and images.
