# Framer Motion: Creating Smooth Animations

Framer Motion is a production-ready motion library for React that makes it easy to create fluid animations and interactions.

## Why Framer Motion?

- **Simple API**: Easy to learn and use
- **Powerful**: Handles complex animations
- **Performant**: Uses hardware acceleration
- **Declarative**: Define animations in JSX

## Basic Animations

Start with simple animations:

```jsx
import { motion } from 'framer-motion'

function Component() {
  return (
    <motion.div
      initial={{ opacity: 0, y: 20 }}
      animate={{ opacity: 1, y: 0 }}
      transition={{ duration: 0.5 }}
    >
      Animated content
    </motion.div>
  )
}
```
